# Chips Overflow Component

Компонент списка чипсов с автоматическим переносом лишних элементов в popover.

## Технологии
- React
- TypeScript
- Vite
- CSS Modules

## Возможности
- отображение чипсов в одну строку
- адаптивный расчет количества
- overflow в popover
- выбор чипсов
- переиспользуемый компонент Chip

## Запуск

npm install
npm run dev
